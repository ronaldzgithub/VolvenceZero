## 一、破：当前 AI 的"工具困境"

当今最先进的 AI 系统，无论参数规模如何膨胀，在结构上仍更接近"高速回放工具"而非自主智能体。这一判断基于两个结构性缺陷：

**其一，时间维度的僵死。** 以全量预训练为周期的更新节奏，意味着模型对世界的认知是"快照级"的。在它被部署的那一刻，它与流动的现实之间就开始累积偏差，且不会在交互中自动收敛。

**其二，认知闭环的断裂。** 从目标设定、环境感知、行动决策到结果反馈，这一完整的认知环路，至今主要由人类在外围兜底。机器擅长"感知—决策"的前半程，"反馈—修正"的后半程很少被真正纳入系统内部、并跨会话固化。

我们的工作假设是：**没有闭环，就没有真正意义上的智能体性（agency）。** 这是我们要验证的命题，不是已被普遍接受的定论。

---

## 二、立：在线持续学习是我们押注的通路

要跨越上述天堑，AI 系统需要走向"在线持续学习"——能够在与环境交互的时间尺度上接收反馈信号，并完成对反馈的"再反馈"：根据行动后果增量修正自身的认知结构，而不必重训整个基座。

这直指智能最底层的三道难题，也是本领域公认的硬骨头：

- **稀疏数据（Sparsity）**——真实环境中有意义的反馈信号极为稀少。系统必须能从少数交互中提取足以改变认知的信息量，而非依赖海量样本的统计平均。
- **抽象能力（Abstraction）**——原始信号是混沌的；智能的关键在于把它压缩为可组合、可迁移的结构化表征。没有抽象，就难有泛化。
- **灾难性遗忘（Catastrophic Forgetting）**——这是在线学习最凶险的障碍。当新经验流入时，如何保证旧认知结构不被覆盖？任何声称"持续学习"的系统都必须正面回答。

> **诚实边界**：我们不声称"在线持续学习是 AGI 的唯一通路"。这是我们经过取舍后**重点押注**的方向，且把它当作**待验证假设**——内部已写明明确的"杀死条件"（见第三部分与 `human-world-model-thesis`）。

## 三、证：理论地基与验证范式

Volvence 的回答不是"大算力暴力碾压"，而是把"低数据下的持续学习"当成一个**在线约束下的泛化问题**来求解。我们的地基有三层：可核验的学术理论、明确分工的架构、以及"最小闭环 ↔ 理论"互相反哺的验证范式。

### 3.1 主动学习理论支柱：杨柳博士（可公开核验）

在"从极少样本里学到最多、并优先补最关键知识缺口"这一问题上，我们有该领域的世界级理论家——**杨柳博士**（CMU 博士，师从 Avrim Blum、Jaime Carbonell）。其代表性成果横跨 2011–2025，发表于 JMLR、Annals of Statistics、NeurIPS、AISTATS、ALT 等机器学习与数理统计顶刊顶会（含多篇第一/唯一作者）：

| 代表性贡献 | 发表（年份/等级） | 严格表述 |
|---|---|---|
| **星数（star number）** | JMLR（2015，CCF-A 顶刊） | 提出刻画低噪声下主动学习 minimax 标注复杂度的组合复杂度度量，并证明此前文献中几乎所有复杂度度量的最坏取值都等于星数 |
| **Active Apprenticeship Learning + Eluder-star 维度** | ALT（2025） | 首创将主动学习与强化学习中的学徒学习相结合的新方向；提出 Eluder-star 维度刻画其样本复杂度；在 Massart / Tsybakov 噪声下给出鲁棒算法及样本复杂度上下界 |
| **可识别混合模型下的主动学习** | Annals of Statistics（在审，第一作者） | 推导满足正则条件的可识别混合模型下，主动学习算法风险的一般上界 |
| **在线选择性采样的错误/查询权衡** | AISTATS（2021） | 建立在线选择性采样的一般理论框架，刻画"预测错误数"与"查询数"之间的最优权衡曲线 |
| **批量主动学习（Buy-in-Bulk）** | NeurIPS（2013，CCF-A，第一作者） | 给出按批查询标签的主动学习标注复杂度界，以及总查询数与轮数之间的权衡 |
| **漂移分布下的主动学习** | NeurIPS（2011，CCF-A，唯一作者） | 在样本分布随时间变化的流式设定下，给出主动学习错误数/查询数上界与 minimax 下界 |

这些工作精确对应 Volvence 必须解决的真实问题：样本极少 → 标注/查询复杂度理论；分布会漂移（用户随时间变化）→ 漂移分布下的主动学习；必须分轻重 → 选择性采样与最优查询权衡；要与强化结合 → Active Apprenticeship Learning。

> **严谨边界**：以上为杨柳博士在学术文献中可独立核验的研究成果（理论方向论文作者多按姓氏字母排序，上表已标注第一/唯一作者者）。Volvence 的工程实现是在这些理论指导下的**独立工作**，二者不等同、不混淆。

### 3.2 抗遗忘与持续适应：架构而非单一技巧

如何"既不忘记昨天，又能理解此刻"，我们的回答是**分层架构上的更新率分离**，而非押注某一种权重正则化技巧：

- **冻结稳定基底（R2）**：取现成强开源权重模型并冻结使用，不在运行时对整模做端到端梯度更新——这从根本上避免了在线学习污染基座先验。
- **只训自适应控制器层（R3/R4）**：元控制器（`z_t / β_t`）在潜空间压缩动作/搜索空间，长期策略学习发生在控制器空间，而**不**压在自然语言输出上（避免奖励投机、推理污染）。
- **多时间尺度记忆连续谱（R5/R6, CMS）**：瞬态/情景/持久/派生索引各自独立 owner，新经验增量写入而非覆盖旧结构——这是系统对抗灾难性遗忘的工程主路径。
- **预测误差作为一级信号（R-PE）**：用 PE 决定下一步问什么、记什么、学什么，实现"反馈—再反馈"的自排课程。

> **诚实边界**：业内的"正交权重调制（OWM）"等正则化方法是抗遗忘的一类思路，但**不是** Volvence 当前的核心机制，也不是上述任何理论成果的内容；我们的抗遗忘主路径是"冻结基底 + 控制器层 + 多时间尺度记忆"。对外不应把二者混为一谈。

### 3.3 认知科学侧翼：测量方法学（评估/接洽中，非已落地）

我们正在评估引入认知科学的实证测量方法学，对象是音乐认知奠基人 **Carol L. Krumhansl** 学术谱系。其价值在于一套"把主观情绪变成逐秒可测信号"的范式（连续实时张力标注 + 生理对齐 + 跨人稳定），可改造为我们 PE / 情绪 / 关系状态 ground truth 的标注与验证工具。

> **严谨边界**：截至本文撰写时点，这是**人才/顾问接洽与可行性评估阶段**，尚无具体人员加盟的既成事实（评估框架见 `krumhansl-cognitive-fit-2026-06.md`）。迁移到我们是**原理同构**（统计学习/期望/PE/多时间尺度/实时测量），**非方法直搬**；工程化成我们的标注/评估工具仍有真实工作量。对外叙事不得把"评估中"讲成"已加盟/已落地"。

### 3.4 学术背书（推进中）

我们正推进与学习理论学术网络（CMU / Yale / Steve Hanneke 系等）的第三方背书与评估合作，目标是把内部 benchmark 升级为可被同行验证的标准。

> **严谨边界**：这是**推进中的目标**（见 `docs/known-debts.md` 相关条目），不是既成的"长期合作认同"。任何具体合作关系/学术职务在拿到可引用证据前，只表述为"接洽/推进中"。

### 3.5 验证范式：理论指导最小闭环，最小闭环反哺理论

我们不走"先理论后大工程"，而是让最小闭环与理论互相反哺。当前阶段的**最小证明目标**是在人类个体化建模任务上做受控 ablation：在相同冻结基底下，验证 **PE + 主动学习 + 多时间尺度控制器** 是否显著优于"普通 LLM + memory / RAG / LoRA / 随机采样标注" baseline。

> **诚实边界**：上述"反馈—调制—不遗忘"的完整在线回路，目前是**待验证假设 + 早期受控实验**，**不是**"已在动态物理仿真 / 实时交互式环境跑通完整回路"的既成事实。内部明确写有杀死条件：若主动学习不优于随机采样、控制器不优于普通 memory/RAG、或 PE 信号无法稳定定义，则 thesis 收缩为产品型 memory 公司。

## 四、愿：从认知基座到数字生命

唯有把闭环咬合好，我们才可能逐步逼近 Rich Sutton 一脉对"经验与目标"的期许：让机器不只是拟合静态数据，而是在时间线上"经历"世界。

我们路线图上有一个清晰的、**待验证的**技术判据（"奇点刻度"，作为未来里程碑而非当前成果）：

> 当系统能在很大程度未知的连续环境中，仅凭稀疏交互（目标量级 < 10 次有效反馈）自主重构环境的因果结构，并能将该结构迁移至一个不相关的新环境时，我们认为"在线认知闭环"在该受限意义上成立。

> **诚实边界**：这是**研究方向上的判据/北极星**，不是已达成的指标，也区别于本轮的**商业验证里程碑**（18 个月：早期留存/付费、PE+主动学习 ablation 证据、物理侧扩张门槛——见 `human-world-model-thesis`）。两者不可混为一谈。

在这一认知基座之上，"数字生命"才可能从隐喻走向工程：机器拥有基于时间连续性与记忆积淀的"情商"，在人机之间建立可信任、可审计、可治理的情感关系。这种能力若能泛化，才可能在现有语言模型与世界模型之上，生长出更具意图性、连续性与自洽性的通用智能。

---

## 结语

过去几年，主流叙事在解决"大"的问题——大算力、大数据。

Volvence 想正面解决一个"活"的问题：**如何让系统既不忘记昨天，又能理解此刻。**

我们把"智能"从一个哲学概念，尝试降维成一个**在线约束条件下的泛化问题**：稀疏、抽象、不遗忘。

只要这个问题在受限场景里被证明可解——而我们用可核验的理论、可回滚的架构和可证伪的 ablation 来逼近它——AGI 就更接近一个工程问题，而非信仰。
