### 三条不变量

1. **IQ 借力，不卷基底**：我们不参加百模大战。把 GPT-5 / Claude 4.7 / DeepSeek V4 等最强基础模型当作冻结基底（Substrate），通过多时间尺度学习（NL）+ 抽象行动复用（ETA）形成大模型自身做不到的"长程关系智商"，享受大厂红利、不付训练账单：

   $$\text{IQ}_{\text{volvence}} = \text{Substrate} \times \text{ETA reuse} \times \text{NL accumulation}$$

2. **EQ 是运行时契约，不是 Prompt 标签**：类人情商在我们的系统里是**契约级不变量**，不是"请温柔说话"那种 prompt 装饰：
   - **双轨学习**：世界模型 PE 与关系模型 PE 物理隔离，用户偏好与边界不会被任务奖励静默污染。
   - **9 类语义 Owner**：plan_intent / commitment / open_loop / user_model / belief_assumption / relationship_state / goal_value / boundary_consent / execution_result，各自独立 owner，只通过不可变快照交互。
   - **Rupture-to-Repair**：用户显式反感 → 自动记录 Rupture → 切 Regime → 后继 30 天 Monetization Suppression。这是行为变化，不是道歉文案。

3. **基底冻结、控制器可塑**：在线学习只发生在低维 latent 控制器（`z_t / β_t`）上；基底模型权重永远不被在线端到端反传梯度污染。所有自修改走 `ModificationGate` 并可回滚（R15）。

> **在电商私域里，Volvence 不是"智能客服"也不是"自动导购"，是一个长期经营关系的 AI 运营伙伴（Relationship Operating Agent）。我们卖的不是回复条数，是关系连续性带来的 LTV 增幅。**

### 痛点拆解

电商私域今天真正的痛点不是"缺群发工具"，而是：**人太多、关系太长、状态太细，人工运营做不到对每个用户保持长期记忆、克制与时机判断。**

| 阶段 | 传统 SCRM / 导购机器人 | Volvence（Mobi 实例） |
|---|---|---|
| Day 1 用户提问 | 打标签 → 入分组 | 记 Context + 写入 user_model owner |
| Day 10 用户反感 | 标签不会衰减；继续群发 | 系统检测 Rupture → 切 Regime → 销售推荐闸门关闭 30 天 |
| Day 30 用户状态变化 | 无机制感知；继续按旧标签推 | 慢反思层从 raw 流式互动中提取"孩子新阶段"信号 + 更新 Belief |
| Day 45 推荐时机 | 仍是定时群发 | 闸门重新开启 + latent 控制器选择克制语气 → 关系内成交 |

### 商业价值：优化关系 LTV，不是省回复成本

我们卖的不是"一条回复省 5 分钱"，而是**关系连续性创造的 LTV 增幅**。

- **Mobi 联营私域（已签 JV，2800 万粉丝池）**：传统人工 + SCRM 群发在规模下的转化率瓶颈 **0.3%**（运营记不住每个人的状态和边界）。Volvence 目标 90 天将转化率提升到 **0.6%–1.0%**。
- **高盖伦母婴（已签 JV，1100 万母婴 / 早教粉丝）**：12–18 个月亲子交互沉淀为 **biographical switching cost**（自传式迁移成本）。父母用 6 个月后，AI 记住孩子所有细碎成长轨迹和父母教育边界；迁移到其他产品 = 关系上下文全部清零。

### 一句话产品定义

> **Volvence = 把"长程关系"做成一份可被运行时审计、跨会话不丢、按 Scope 隔离、按 Regime 克制的资产。私域不是用户群，私域是关系账本。**

### 关系治理的 7 个硬维度（moat 所在）

当大厂把 Memory 做到无限长时，只有 Volvence 在 runtime 严肃解决以下 7 个商业 / 法律现实问题：

1. **什么该跨会话持续？** 用户过敏史 / 教育底线不能随会话重置遗忘。
2. **什么会按时间自然衰减？** 6 个月前孩子的睡眠习惯到 12 个月必须衰减，否则干扰决策。
3. **什么永远不能迁移或越界？** Alice 的偏好与隐私在多用户 / 家庭进程里物理隔离，RAG 不能越界召回。
4. **什么在什么时间尺度上自适应？** 即时语气 turn 级；性格 / 关系体制 Session 级或 background 异步。
5. **何时必须自动中止商业变现？** Rupture 一触发，即便大模型预测此刻推销 80% 成交，runtime 也强制拦截。
6. **关系破裂后如何真正修复？** 不是道歉模板，是后续 3–5 轮交互中 latent 控制向量切换，给出用户可感知的行为克制。
7. **长程同意 / 删除 / 法规审计如何变成产品能力？** 一键 Delete-Me + 物理擦除 + 可被法庭采信的审计证据，对齐 EU AI Act / PIPL / GDPR Art. 17。


### 8 段链条

1. **下一代 AGI 首先是认知 AGI（Cognitive AGI）**——不是更大的静态预训练 world model（那只是放映机），也不是机械本体（那只是木偶）。真正的 AGI 是在真实长程交互中形成自主目标、记忆、抽象、边界适应与主动信任修复的认知系统。
   - *证：Sutton & Silver (DeepMind 2024)《The Era of Experience》——人类数据预训练见顶，经验流式学习是下一 paradigm。*

2. **Cognitive AGI 必须在神经网络上在线持续学习（Continual Learning）**——手写 Prompt 工程、向量数据库、上下文拼凑、Agent Workflow 本质上是 1980 年代专家系统复刻，堆积无法 debug 的智慧债，必将在"苦涩的教训"面前再次崩溃。
   - *证：Sutton (2019)《The Bitter Lesson》；2024-2025 年 Agent 拼装产品在长程生产环境的大面积雪崩。*

3. **人类长期互动轨迹是唯一不可逆的垂直数据护城河**——公共互联网 text 数据 2026-2032 见顶；机构数据（Mayo Clinic 等）可枚举或被授权；最庞大、可再生、无法逆向生成、具生物学排他性的，是人每天产生的真实长程关系数据。
   - *证：Epoch AI 2024；Mayo Clinic 数据壁垒模型的延伸。*

4. **持续学习的核心必须是 RL，但直接在 Token 空间做 RL 在结构上已被证明灾难性**——直接在自然语言 token 或 CoT 上施加强化压力，不可避免诱发对外部对齐的伪装（alignment-faking）、推理链自发污染（CoT obfuscation）、内部决策与外部推理的背离。
   - *证：Anthropic 2025 / OpenAI 2026 数篇关于 Reasoning Model 滥用 Token-RL 导致失控与对齐作假的论文。*

5. **可行的 RL 必须发生在系统涌现的低维 latent 行动空间（`z_t`）上，并按多时间尺度分层**——慢反思 meta-learn 快层初始化；切换边界 `β_t` 在 latent 空间学习优于 token 空间。
   - *证：Google Research《Nested Learning》(arXiv:2512.24695, 2025) 多频层协同初始化；ETH《Emergent Temporal Abstractions》(arXiv:2512.20605, 2025) latent 切换边界优于 token 空间。*

6. **Volvence 针对此学术共识给出三件套底层闭环解法**：
   - **(A) Reward = Body**：目标不来自外部标注（alignment-faking 的根源），来自内部 Homeostatic Drives（Trust / Empathy / Restraint / KB-share）与预测误差（PE）。*（Botvinick / Wang / Dabney DeepMind 2025 distributional-dopamine 生物学证明；Friston 主动推理。）*
   - **(B) 抽象涌现自 `CMSVariant.NESTED` + ETA**：慢反思层 meta-learn 快层理想初始化；repo 内 meta-learning 收敛在 Task / Composition / Credit-alignment matched-control 实验中全 PASS。
   - **(C) 小数据高样本效率来自主动学习（Active Learning）**：用 Yang Liu Minimax 理论，在不平稳流式 sparse data 下高精度收敛。

7. **"薄 Prompt 厚运行时"的工业级架构已在 repo 完整实现**——
   - 1100+ Contract Tests 守门，5 大垂直生命体在单进程内 CI 物理隔离，完全解耦大模型；任何自修改走 `ModificationGate` + 快照可回滚（R15）。
   - 跨模块只走不可变快照（R8）；运行时控制器不能静默成为基底数据的第二所有者。

8. **这是一条不与通用大厂竞争、独立自足、并享受大厂红利的商业闭环**——
   - 大厂卷 IQ 底座，底座越强越便宜，我们的 substrate 成本越低；
   - 我们通过已签 6 个 JV，垂直且排他地掌控 45M 用户的关系轨迹数据，筑起 Relationship Governance 的终极 moat。
