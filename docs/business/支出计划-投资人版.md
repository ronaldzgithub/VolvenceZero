# Volvence 2026–2028 支出与运营计划（投资人版）

**文档性质**：对外提交主稿。客观陈述未来三年的支出与运营计划，不为说服。  
**口径基准**：本计划与 [docs/business/支出计划.md](支出计划.md)（v2.1 工作底稿）数字一致；本文件去除内部勾稽、版本演进与双口径备注，便于快速通读。  
**货币**：人民币（万元）。  
**会计口径**：净利润为会计净利润（CapEx 资本化、3 年直线折旧入损益）。损益**不含**股份支付费用——见 §7。

## 2. 收入结构与商业逻辑

### 2.1 三条收入线

| 来源 | 2026 | 2027 | 2028 |
|------|----:|----:|----:|
| 合资公司股权 / 分成 / 联合商业化 | 2,165 | 9,000 | 16,000 |
| API / token / 平台服务费 | 1,140 | 8,100 | 27,000 |
| 版本开发费 / POC / 其他技术服务 | 495 | 900 | 2,000 |
| **合计** | **3,800** | **18,000** | **45,000** |

> "合资公司"列含合资主体支付的所有款项（分红 + 股权 + 联合商业化 + 主体内技术服务费分成）；"API / 平台"列**仅含**合资主体之外的客户（直签 API / 自助平台 / 第三方调用 / 海外）。两列互斥不重叠。

### 2.2 合资公司：12 → 16 → 20 家

| 年度 | 累计家数 | 同比新增 | 单家年均贡献 | 向稳态收敛 |
|------|--------:|--------:|------------:|----------:|
| 2026 | 12 | +12 | ≈ 180 | 18% |
| 2027 | 16 | +4 | ≈ 563 | 57% |
| 2028 | 20 | +4 | = 800 | 82% |

> 单家稳态约 980 万元 / 年，参考 [docs/business/合资公司资金收入测算.md](合资公司资金收入测算.md)。2027 / 2028 同比新增仅 +4 家，组织扩张主要服务于**已签约存量达产**而非家数线性放大。

### 2.3 API / 平台收入增长逻辑

API / 平台收入三年扩张倍率：1,140 → 8,100 → 27,000（**2027 同比 +611% / 2028 同比 +233%**）。该曲线**不依赖等比例放大销售队伍**，主要由两个网络效应驱动：

- **已有装机量的消耗**：2026 完成 12 家合资公司落地后，每家本身即是大流量入口与产业链支点；其 C 端用户、合作伙伴、上下游客户的 API 调用量随业务自然消耗持续放大，新接入边际获客成本接近零。
- **网络级裂变与传播**：成功案例形成口碑外溢与跨场景集成（SaaS 间互调、数据飞轮、内容生态），合资 / 平台客户彼此引荐与互调，形成多对多的接入网络。

与组织设计一致——商务 / BD 自有人头占运营编制 50% 上下，不为线性收入增长配等比例销售队伍。

## 5. 融资节奏

| 轮次 | 募资区间 | 主要用途 |
|------|---------|---------|
| 天使轮 | **690 万** | 引擎稳定化、首批商业化、组织扩编（含 2026 年 250 万 CapEx 启动部分） |
| A 轮 | **1.2–1.5 亿元** | 覆盖 2027 年 4,000 万 CapEx、合资规模化交付、平台与 API、海外试点；预留 6–9 个月运营缓冲 |
| B 轮 | **2.5–4.0 亿元** | 覆盖 2028 年 8,000 万 CapEx、平台化深化与全球交付；与经营回款协同 |

**原则**：单轮在区间内随谈判取值，不为叙事抬升募资额；下轮融资若滞后，支出按优先级收缩——优先已签约交付与合规，其次缩减扩张性 CapEx 与市场费用。

## 7. 关键风险与口径声明（直陈）

- **股份支付口径**：本计划损益**不含**股份支付费用。2026 期权池尚未启动、无影响。**2027 年起**若审计要求按 IFRS 2 / CN GAAP 11 号将股份支付入损益（按 A 轮估值 5 亿、10% 期权池、4 年归属粗算约 1,250 万 / 年），可能影响协议利润达成。**建议在补充协议中预先明确利润口径**——业内惯例以"扣非后净利润 + 加回股份支付"作为对赌基准。

- **2027 经营现金流 ≈ CapEx，0 净结余**：2027 经营现金流约 4,000 万 = 当年 CapEx 4,000 万，意味着年内无自由结余。任何关键变量（合资爬坡、API 增长、A 轮到位时点）的小幅延迟都会立即占用 A 轮缓冲。

- **API 增速敏感性**：2027 协议安全垫看似宽（+1,083 万），但若 API 同比从 +611% 降至约 +400%，2027 净利润将贴近协议下限。所述 +611% / +233% 增长依赖 §2.3 两个网络效应能按预期释放。

- **CapEx 中可资本化研发的会计认定**：2027 / 2028 无形资产中含可资本化研发 2,000 / 4,000 万。若审计要求部分研究阶段费用化，会侵蚀当年净利润；保守口径下三年净利润仍达成协议下限。

- **2026 节奏**：商业化与大规模交付自 5 月启动，1–4 月为预备期；损益按完整年度列示，但月度现金流低点出现在 5–6 月（回款滞后于支出），需以天使轮支撑。

## 8. 执行原则

- **融资规模与 CapEx 节奏直接挂钩**，不以估值叙事抬升募资额；安全垫为运营缓冲、非利润粉饰。
- **支出按优先级收缩**：已签约交付与合规优先；扩张性 CapEx、市场费用、非紧急研发可分阶段后置。
- **组织扩张严守"人头增速远低于收入增速"**：2026 → 2028 收入增长约 11.8 倍，编制增长 5 倍，平台与网络效应承担差额。
- **协议利润口径前置约定**：股份支付、可资本化研发的会计处理在补充协议中预先约定，避免后续审计意见触发歧义。
