>
> **Source corpus:**
>
> - Villalobos, Sevilla et al., *Will We Run Out of Data?* (Epoch AI, 2024) — internet text ceiling 2026–2032
> - Open Evidence (Xfund portfolio) — institutional vertical data analogue
> - Per-user volume estimate — 10 MB/day per user, public-figure approximation for chat + voice + structured state
>
> **The implication:** any frontier lab that runs out of net-new pretraining text by 2030 has *only two* options to continue improving: synthetic data (limited fidelity) or **per-person interaction data** — which requires being the runtime owner of the user relationship, which requires the architecture this deck describes.

**Speaker script (2 min)**

The data ceiling is the second piece of why the thesis is investable now, not in five years.

Public-internet text is closing its ceiling between 2026 and 2032 — Epoch AI's projection, not ours. Institutional vertical data is finite — Mayo Clinic is enormously valuable but you can enumerate the next twenty institutional sources. The third row is where the asset class is.

A billion users at ten megabytes per day of interaction is 3.6 exabytes per year. That is roughly five orders of magnitude larger than indexed internet text. Crucially, it is *renewable* — every day produces new data — and *non-transferable* because consent and history live with the runtime that owns the user relationship.

The implication for the frontier-lab competitive position: any lab that runs out of net-new pretraining text by 2030 has only two real choices to keep improving. Synthetic data — limited fidelity. Or per-person interaction data — which requires being the runtime owner, which requires the governance architecture this deck describes. **This is why our substrate gets cheaper as foundation labs get more desperate for data, not more expensive.**
