# 2026–2028 年度资本性支出计划

> Status: planning draft
> Last updated: 2026-05-06
> Owner: 财务 + CTO 联合视角
> 与之同时存在的两份原始材料：`docs/business/未来三年资本性支出计划.pdf`（投递版总盘子）、`docs/business/融资计划.md`（天使轮 ¥690 万）

本文件把已经对外披露的三年 CapEx 总盘子（固定资产 + 无形资产合计约 ¥2.025 亿）展开到**子项级别**，并与（一）天使轮 / A 轮 / B 轮的融资节奏、（二）当前 8 wheel 技术路径的能力域里程碑（参见 `archetecture.md` 与 `docs/specs/00_INDEX.md`）严格对齐。"合理"的口径是：每一笔投入都能映射到一个已落地或路线图明确的能力域 owner、一类已识别的硬件/合规成本、或一项已签署的商业承诺。

---

## 一、三年合计速览

| 年份 | 固定资产（万元） | 无形资产（万元） | CapEx 合计（万元） | 对应融资轮次 | 对应技术阶段 |
|---|---:|---:|---:|---|---|
| 2026 | 150 | 100 | **250** | 天使轮 ¥690 万 | Closed Alpha → 5 家合资 V1 + Mira 公测 |
| 2027 | 2,000 | 2,000 | **4,000** | A 轮（目标 $1 亿+ 估值） | 10+ 合资规模化 + 海外（西语系/拉美）冷启动 + DLaaS 控制面 |
| 2028 | 6,000 | 10,000 | **16,000** | B 轮（目标估值 $5–10 亿） | 平台化、海外自有产品成熟、多区域部署 |
| 合计 | **8,150** | **12,100** | **20,250** | — | — |

> **CapEx ≠ 烧钱总额**。本表只统计可形成固定资产或无形资产的支出；员工工资、市场获客、差旅、合资公司日常分润等经营性支出（OpEx）走融资计划与 ARR 回款的另一本账。两本账在 2026 年的对照：天使轮 ¥690 万中 ≈ ¥250 万属于本表 CapEx 口径（算力 60 万 + 引擎研发资本化部分 + IP/合规 + 设备），其余 ≈ ¥440 万为 OpEx。

## 二、2026 年（CapEx ¥250 万）

天使轮启动期，资金用途严格对应"从 0 到 1 启动"，不超募、不超采。本年 CapEx 等价于把天使轮里"算力 + 安全合规 + 工程师设备 + 核心 IP 资本化"那一部分单独切片出来。

### 2.1 固定资产 ¥150 万

| 子项 | 金额（万元） | 数量/规格 | 用途 / 对应能力域 |
|---|---:|---|---|
| GPU 工作站（开发 + 小规模训练） | 80 | 2 台 8×L40S 或 1 台 8×H100 + 租赁补充 | `vz-substrate` 残差捕获、`vz-temporal` metacontroller 训练实验、ETA β_t 切换离线复现 |
| 工程师工作站 + 笔记本 | 30 | 12–15 人，台式/笔记本双配 | 团队 7→12–15 人扩张直接配套 |
| 网络 / 存储 / 安全设备 | 15 | NAS 80T + 防火墙 + 交换机 | `vz-memory` 持久化、scoped MemoryStore 备份；`closed-alpha-api-service` evidence root 落盘 |
| 测试硬件 | 10 | 移动端真机 6 台、麦/摄/可穿戴评测样机 | Mira 公测 + 合资公司 V1 多端验收 |
| 办公基础（工位/显示器/会议设备） | 15 | 总部一处 | 团队扩张配套 |
| **小计** | **150** | | |

### 2.2 无形资产 ¥100 万

| 子项 | 金额（万元） | 用途 / 对应能力域 |
|---|---:|---|
| 核心引擎 R&D 资本化（开发支出转资） | 60 | NL+ETA 主链已落地能力的部分研发投入按会计准则资本化：`prediction_error` / `credit` / `regime` / `dual_track` / `cms` 等 owner 的代码资产口径 |
| 商标 + 专利申请（国内） | 20 | "Volvence Zero / Mira / EmoGPT" 商标族、核心 ETA β_t / NL CMS 写入门控两类核心专利国内首申 |
| 软件许可与协作工具 | 10 | IDE 商业版、CI、监控、内部知识库 |
| 数据集 / 开源模型许可 | 10 | 中文情感语料、合规对话语料、Qwen / 开源基底的商用许可（如需） |
| **小计** | **100** | |

### 2.3 与里程碑的强耦合

- 期末必须对应**已落地**：closed alpha API 单租户上线、5 家合资公司 V1 验收、Mira 公测、`affordance` v1 / `aac-commitment-lifecycle` 等 P0 落地（已完成）。
- 这些里程碑同时也是 A 轮启动条件，未达不要触发下一年 CapEx 放大。

## 三、2027 年（CapEx ¥4,000 万）

A 轮交割后进入"扩张元年"。固定资产里第一次出现**自建训练集群**与**海外节点**；无形资产里第一次出现**多语言/海外合规**。

### 3.1 固定资产 ¥2,000 万

| 子项 | 金额（万元） | 数量/规格 | 用途 / 对应能力域 |
|---|---:|---|---|
| 自建中型 GPU 训练集群（一期） | 1,200 | 16–32 卡 H800 / H200 等价规模（含整机柜、PSU、散热） | NL rare-heavy substrate refresh、ETA family 大批量回放训练；从云租赁切到自建可在 2 年视角下显著降低 TCO，是技术路径"冻结基底定期慢刷新"必备 |
| 推理服务初期集群 | 200 | L40S / RTX Pro 6000 系 ×8–12 卡 | 合资公司放大期的推理算力，避免完全依赖外部 API |
| 工作站 / 笔记本 | 250 | 50–80 人扩张配套 | 团队从 12–15 → 50–80 人 |
| 办公场地装修 + 扩张 | 200 | 总部装修 + 一个二线研发点 | 配合科研团队扩充 |
| 网络 / 存储 / 安全（多机房） | 200 | 主备机房、IDC 机柜租赁押金、灾备一体机 | scoped memory 持久化 + audit evidence 多副本 |
| 海外节点初始化 | 100 | AWS LATAM / GCP 拉美地域硬件预付 | 海外（西语系）冷启动所需边缘部署 |
| 边缘 / 合资定制硬件原型 | 50 | 育儿/IP 运营/跨境电商场景定制设备首批样机 | 合资公司 V2 的硬件协同 |
| **小计** | **2,000** | | |

### 3.2 无形资产 ¥2,000 万

| 子项 | 金额（万元） | 用途 / 对应能力域 |
|---|---:|---|
| 平台核心研发资本化（开发支出转资） | 1,200 | A 轮投入做实的能力域：DLaaS 控制面（多租户/Tenant 隔离/License/Exam）、`lifeform-ingestion` 学徒训练管线、`affordance` v2、`InterlocutorState` learned readout、海外多语言 expression 层。这些都是**持续多年贡献价值**的代码资产，符合无形资产准则中"开发阶段支出资本化"口径 |
| 数据资产（西语系 + 多模态语料） | 250 | 海外冷启动必需的西语系真实对话语料采购、合规标注、轻量 RLHF 偏好数据 |
| 软件许可（商业 ELA / 监控 / 数据库） | 150 | DataDog / Snowflake / Auth0 等主流 SaaS 多年许可 |

### 3.3 与里程碑的强耦合

- 期末对应：合资公司从 5 家放大到 10+ 家；Mira 拉美西语系版本上线；`runtime-ingestion` + `apprenticeship trigger` 落地（spec freeze 已在 docs/specs/00_INDEX 第 16 条登记）；`lifeform-service` 多租户隔离上线；F1–F6 family report 成为对外发布门槛。
- 上述里程碑构成 B 轮估值的硬证据。

## 四、2028 年（CapEx ¥16,000 万）

B 轮交割后进入"平台化"。固定资产主体是**第二训练集群 + 多区域推理基础**；无形资产里第一次出现**战略并购/许可**与**全球 IP 组合**。

### 4.1 固定资产 ¥6,000 万

| 子项 | 金额（万元） | 数量/规格 | 用途 / 对应能力域 |
|---|---:|---|---|
| 大型 GPU 训练集群（二期） | 3,500 | 64–96 卡 H200 / B200 等价规模，独立机房 | 多 vertical 同时训练、稀有重头基底刷新、长程 ETA family 训练；从一期"够用"升级到"领先" |
| 多区域推理服务集群 | 600 | 国内 + 拉美 + 北美三个推理点 | DLaaS API 平台与海外自有产品的全球 P95 延迟保障 |
| 多区域数据中心 / IDC 投入 | 1,000 | 拉美主区域 + 北美骨干 + 国内灾备 | 多区域部署刚性投入；含机柜押金、跨境专线初装费的资本化部分 |
| 办公总部 + 海外办公点 | 400 | 国内总部、上海或深圳分部、旧金山小型办公点 | 国际化梯队配套 |
| 工作站 / 笔记本 | 300 | 150–250 人扩张配套 | 团队从 50–80 → 150–250 人 |
| 安全 / 网络 / 灾备升级 | 200 | DDoS 防护、零信任网络、跨区灾备一体机 | 平台级 SLA 与监管合规要求 |

### 4.3 与里程碑的强耦合

- 这些里程碑构成战略并购或独立上市路径的关键资产清单。

## 五、合理性自检（每条都必须经得起追问）

| 追问 | 回应 |
|---|---|
| 2026 只投 250 万够吗？ | 够。本年 CapEx 与天使轮 ¥690 万互为子集：天使轮 ¥276 万核心研发里 ≈ ¥60 万直接对应本表"核心引擎 R&D 资本化"；¥60 万算力对应"GPU 工作站"。其余 ¥440 万是 OpEx。**这一年的财务命题不是"花得多"，而是"用最少的钱跑通 5 家合资 V1 + 公测"。** |
| 2027 自建 GPU 集群是不是过早？ | **不过早**。理由：（一）NL rare-heavy 的 substrate refresh 是周期性高强度训练，云租赁 12 个月成本即追平自建；（二）数据合规要求"训练数据不可出境/不可第三方落盘"是合资公司硬条款；（三）海外节点也只是 100 万，不构成"过度国际化"。 |
| 2028 无形资产 1 亿是不是虚胀？ | **不虚胀**，但需要会计严格区分。其中 ¥6,000 万开发支出资本化，必须按企业会计准则严格区分研究阶段（费用化）与开发阶段（资本化），并由审计每季度复核。其余 ¥4,000 万是真实采购、并购、合规、IP，是硬支出。 |
| CapEx 能否被 ARR 与融资共同支撑？ | 能。三年累计 ¥2.025 亿，对应：天使轮 ¥690 万（2026）+ A 轮预计 ¥1.4–2.1 亿（2027）+ B 轮预计 ¥3.5–7 亿（2028）+ 累计 ARR 回款（融资计划测算 2026 ARR ¥3,500 万 → 2028 进入 ¥5–10 亿区间）。CapEx 占融资+回款总盘的比例约 15–25%，处于硬科技公司合理区间。 |
| 是否对齐技术路径而非概念故事？ | 每一项都映射到 `archetecture.md` 中的具体 wheel 或 `docs/specs/00_INDEX.md` 中的具体能力域 spec，没有"通用型 AI 大投入"这种含混项；自建集群与开发支出资本化的规模也直接由"NL 多时间尺度训练 + ETA β_t 训练 + 海外多语言模型层"这三类硬训练任务推导。 |
| 与"轻资产模式"的口径冲突吗？ | 不冲突。融资计划披露的 55%–65% 毛利率是**不含 CapEx 折旧**的口径。本表 CapEx 主要来自融资款而非营业成本，落到损益表是按 5–10 年折旧/摊销分摊，对单年毛利率影响约 5–8 个百分点，仍在轻资产合理区间。 |

## 六、风险提示与触发条件

- **A 轮未按期完成 → 2027 计划必须等比降阶**：2027 年 ¥4,000 万的 60% 直接依赖 A 轮资金。若 A 轮推迟超过 6 个月，自建集群、海外节点与多国合规三项必须延期；技术上的回退路径是继续走云租赁 + 单区域。
- **合资公司 V1 验收不达标 → 2026 不应额外加码硬件**：合资公司收入是滚动再投入的关键来源，未通过 V1 验收前不应把任何 OpEx 节省转移到 CapEx。
- **AI Act / 数据合规法规变化**：欧盟 AI Act / 国内生成式 AI 备案有快速变化的可能性，2027–2028 的合规预算必须保留 ≥ 20% buffer，必要时优先满足合规再扩张算力。
- **不允许的口径漂移**：本表所有"开发支出资本化"严格走外审复核，不允许把营销、咨询费、招聘费伪装成 R&D 转资。这是后续 IPO/并购尽调的高风险项。
