## 各论文一句话定位（10 篇 + 备选若干）

### Top 10 入选

- **2604.18701 Curiosity-Critic**：把瞬时 PE 替换为"PE 的可改进部分"，在线分离 epistemic vs aleatoric — 噪声不再触发 credit / needs / regime 反应；EMA running stats + learned critic 双阶段实现。命中 **R-PE**（直接拆 PE 为两层）。**今天的最强 PE-as-intrinsic-reward**：已在 [`docs/specs/prediction-error-loop.md`](../../../docs/specs/prediction-error-loop.md) Phase 1.B 落地。
- **2507.16598 Depression Distributional Coding**：标量 mean PE 在分布塌缩时丢失关键信号；价值分布从"健康宽分布"塌缩为"窄峰 + 偏侧"是抑郁状态的神经标志；提出 IQR / entropy / asymmetry 三维分布 readout。命中 **R-PE 升级**（PE 升级为带分布形状的 readout）+ **R7 加强**（self-track 健康的 in-vivo 神经背书）。已在 [`docs/specs/prediction-error-loop.md`](../../../docs/specs/prediction-error-loop.md) PE Distributional Readout 上线。
- **2511.22226 Embedded Universal Predictive Intelligence**：把 FEP 重新表达为"嵌入环境的预测智能"，提出 PE-only 学习信号替代外部奖励，最贴近 VZ 的 R-PE 一级信号设计。命中 **R-PE 理论合法性**；R12 反向证据（不应把 evaluation 当学习源）。**今天还要看**：当前最像 R-PE 设计 spec 的论文，写卡后纳入 [`docs/next_gen_emogpt.md`](../../../docs/next_gen_emogpt.md) Sources。
- **2107.12979 PC: Theoretical and Experimental Review (Millidge / Seth / Buckley)**：PC / FEP 当前最权威 arXiv 综述：核心数学结构 + 生物微回路 + 与 BP 的关系 + 与 Kalman 滤波 / normalizing flows 的关系。命中 **R-PE canonical 引用**。**今天还要看**：所有"PE 一级信号"工作绕不开的入门 reference；写 R-PE spec 时必须列入 Sources。
- **2602.23681 ODAR (Active Inference Routing)**：用 amortized active inference + free-energy 在 fast / slow 模型间路由（变分原理形式化的 metacontroller β_t 决策）；EFE = 期望自由能 = epistemic + pragmatic value。命中 **R-PE 加强 + R3 / R4 加强**（β_t 决策的变分化）。**可立刻反哺**：写入 `metacontroller` spec 算法候选。
- **2506.06725 WorldLLM**：用 PE / 不确定性驱动 LLM 主动找证据，把 LLM 当 theory generator + agent 收据；reflection 慢路径的 worldview 修正信号源。命中 **R-PE 加强 + R5 / R6（reflection ⇄ memory）补全**。
- **2412.10425 Active Inference for Self-Organizing Multi-LLM**：FEP 当 cognitive 调度层，自组织协调多 LLM agent；又一次验证 PE / free-energy 是统一信号。命中 **R-PE 加强 + R8（snapshot 跨 LLM 协调）补全**。多 LLM 协调当前不是 VZ 主战场，但思想对齐。
- **2006.04182 PCN ≈ Backprop (Millidge / Tschantz / Buckley)**：证明 PCN 渐近收敛到精确 BP 梯度，且只用 local + Hebbian 规则；扩展到任意 computation graph（CNN / RNN / LSTM）。命中 **R-PE 加强（PE 作为 local credit 信号的硬证据）+ R2 反向证据**（如 PE 等价于 BP，"基底冻结 + 控制器在线 BP"或许有更生物的等价路径）。**今天还要看**：PE 不仅是辅助信号、能完成全部 credit 工作的最强理论证据。
- **1705.05363 ICM (Pathak)**：把好奇心定义为 forward dynamics 在 self-supervised inverse-features 空间上的预测误差；首次让 agent 在无外部奖励的稀疏环境中学习（VizDoom / Mario）。命中 **R-PE 加强**。**今天还要看**：所有"PE → 内禀奖励"工作的源头（包括 #1 Curiosity-Critic 直接 build on）；现代版本如 Curiosity-Critic 都在解决 ICM 的 noisy-TV 问题。
- **1810.12894 RND (Burda / OpenAI)**：用"预测随机网络的固定输出"误差作为 novelty bonus；简单、计算便宜、首次在 Montezuma's Revenge 上超越人类平均（无演示无内部状态）。命中 **R-PE 加强**（最简洁的 novelty PE 实现）。**今天还要看**：现代 NGU / Agent57 都 build on RND；演示了 PE-as-novelty 在大规模深度 RL 上的极简可工作版本。

## 候选→入选筛选规则的几条具体应用

1. **VZ=5 优先**：#1 Curiosity-Critic / #2 Distributional Coding / #3 Embedded UPI / #4 PC Review / #5 ODAR 五篇全部 VZ=5（直接落点 R-PE / R7 接口或 canonical 引用），破格放宽其他维度（#3 Embedded UPI 影响力暂为 2 仍排第 3）。
2. **PE-as-primary 优先**：[`docs/specs/prediction-error-loop.md`](../../../docs/specs/prediction-error-loop.md) 已落地的两条算法（#1 Curiosity-Critic Phase 1.B / #2 PE Distributional Readout）排在前两位，方便精读卡直接引回 spec 段落。
3. **classic 上限 4 篇紧贴**：#4 PC Review / #8 PCN-BP / #9 ICM / #10 RND 四篇全部 ≥ 2017 经典，恰好 4 篇上限；每篇都有"今天还要看"的明确论证（PE-as-intrinsic-reward 的源头 / canonical 入门 / PE 完成 BP 工作的最强证据）。
4. **神经科学 ≤ 1 篇**：#2 Depression Distributional Coding 占该子领域；#11 3M-Progress 列为 backup。
5. **Sussex 系不超过 4**：Millidge / Buckley 共同署名的 #4 + #8 = 2 篇（#6 WorldLLM 一作 INRIA 不计入），未触上限。
6. **token 空间 RL 一律降权**：DeepSeek-R1 风格的 token-RL 在 A1 已收录，B1 不重复；本轴所有候选都在控制器 / 内部表示层（z_t / latent action / 抽象 z 空间），符合 R4 不变量。
7. **新作 ≥ 6 篇**：#1 #2 #3 #5 #6 #7 共 6 篇 ≥ 2024-01，恰好达到下限。
