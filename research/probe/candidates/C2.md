# 候选清单 — 轴 C2 Mechanistic Interpretability & Internal Control

> 时间窗口：2024-01 ~ 2026-05（强制 ≥ 6 篇），辅以奠基经典 ≤ 4 篇。
> 评分协议：[`../01_method_and_scoring.md`](../01_method_and_scoring.md)。
> 本轴 R-ID 关联：**R4**（内部控制器代码空间）、**R8**（SSOT，内部状态可命名）、**R12**（评估只读，禁止反向作 reward）。

## 1. 轴定义与子领域分配

| 子领域 | 覆盖论文 (#) | 备注 |
|---|---|---|
| (a) Sparse autoencoders / dictionary learning | C2-01 / C2-02 / C2-03 | SAE 是 R8 "可命名内部状态"的当前最佳工程化实现 |
| (b) Circuits 经典（IOI / Induction Heads / Greater-than） | C2-10（IOI）+ C2-01 跨覆盖 | 给 R8 提供"内部子图作为 owner"的最早范式 |
| (c) Activation steering / inference-time intervention | C2-04 / C2-05 / C2-07 | 直接对应 R4 控制器代码空间干预 |
| (d) Geometry of concepts（LRH / 真值几何 / Persona / Refusal direction） | C2-05 / C2-07 / C2-09 | 给 R4 + R8 "向量级身份/状态"几何形式化 |
| (e) Function vectors / task vectors | C2-06 | 给 R4 z_t 的"任务即向量"压缩表示 |
| (f) Probing / behavioral probes（Truth / Honesty / Deception probes） | C2-04 跨覆盖（TruthfulQA + 探针头）| 给 R12 "评估只读读出"的方法学 |
| (g) Attribution patching / causal intervention（ROME / MEMIT） | C2-01 跨覆盖（feature-level patching）；ROME/MEMIT 见跳过表 | R10 ModificationGate 反面教材：rank-1 编辑为何不安全 |
| (h) KV cache steering / prompt-level control | C2-08 | R4 之外的"无梯度有界注入口" |

## 6. 选材原则总结（本轴）

1. **每子领域至少 1 篇**（共 8 个子领域 → 在 10 篇内通过跨覆盖完成）：
   - (a) SAE：3 篇（C2-01/02/03）—— 核心方向，多机构独立验证（学术 + Anthropic + DeepMind）
   - (b) Circuits：C2-10 + C2-01 跨覆盖
   - (c) Steering：3 篇（C2-04/05/07）
   - (d) Geometry：C2-05/07/09
   - (e) Function vectors：C2-06
   - (f) Probing：C2-04 跨覆盖
   - (g) Attribution patching：C2-01 跨覆盖（feature-level）；ROME/MEMIT 仅在跳过表
   - (h) KV cache：C2-08

2. **R-ID 覆盖至少各一篇**：
   - R4 内部控制：C2-04/05/06/07/08/09 共 6 篇
   - R8 SSOT 命名：C2-01/02/03/09/10 共 5 篇
   - R12 评估只读 + R10 反向警示：C2-01（feature ablation 评估端）+ C2-05（含 R12 边界冲突点，需 spec 明确） + C2-07（R10 ModificationGate 警示） + C2-09（人格漂移监控只读）

3. **VZ 关联度 = 5 的"立刻反哺"候选**：C2-01 / C2-06 / C2-08 / C2-09（共 4 篇），满足"每轴至少 1 篇 VZ=5"。

4. **Anthropic 配额控制**：仅 C2-02 + C2-09 = 2 篇（≤ 4 限制内有较大余量）。

5. **arxiv 不可下论文**：仅 C2-02 一篇（Transformer Circuits 平台特有）；其余 9 篇均有有效 arxiv id 且已通过 web 搜索验证。
