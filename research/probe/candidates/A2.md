## 各论文一句话定位（10 篇 + 备选若干）

### Top 10 入选

- **2509.24527 Dreamer 4**：先用 unlabeled video + 极少 action label 训 world model（**shortcut forcing objective + efficient transformer** 防 imagination rollout 漂走），再在其内部纯 imagination RL 训 policy；**首个纯 offline 数据下完成 Minecraft 钻石获取**。命中 **R3（latent imagination = z_t 空间想象）/ R4（policy 训在 latent rollout 不在 raw obs）/ R-PE（reconstruction PE 是 imagination 收敛信号）**；与 [`docs/specs/multi-timescale-learning.md`](../../docs/specs/multi-timescale-learning.md) 的 background-slow imagination-based reflection 直接对应。
- **2506.09985 V-JEPA 2**：1M 小时无 action 视频 SSL 预训 1B 编码器；冻结编码器后用 < 62 小时 Droid 机器人数据训 300M 参数 action-conditioned predictor (V-JEPA 2-AC)；新环境 Franka 机械臂 zero-shot 抓取——**这是"冻结视觉基底 + 控制器层适配"哲学的最完整外部实证**。命中 **R2（视觉编码器冻结）/ R3（动作-条件 predictor = z_t 接口）/ R-PE（mask denoising = predictive coding）**。
- **2402.15391 Genie**：11B foundation world model 从纯 unlabeled internet video 完全无监督训练，含 spatiotemporal video tokenizer + autoregressive dynamics + **latent action model**；用户可以对生成的"无穷虚拟世界"做 frame-by-frame 控制，无需 ground-truth action label。**latent action interface** 是 R3/R4 的工程母题——control 不在 token / pixel 而在学到的紧致 latent。
- **2505.13934 RLVR-World**：用可验证奖励训世界模型（state prediction accuracy / task success），绕开 MLE 偏差；多 domain 验证（text game / web nav / robotic manipulation / video）。**直接对应 R-PE → R10 路径**：用 evaluation 信号反训 world model 自身，与 ModificationGate 的"eval 必须先做硬"哲学一致；本调研 #4 作者 Wu Jialong 同时也是 #17 iVideoGPT 一作，体现 Tsinghua WM 系的方法学连续性。
- **1911.08265 MuZero**：在不知规则的前提下，只学 representation / dynamics / prediction 三个网络，预测 reward / value / policy 三个量；MCTS 直接在 latent 上展开。**今天还要看**：所有现代 model-based planning（Stochastic MuZero / EfficientZero / AlphaTensor / AlphaDev）都建立在 MuZero 的"只预测对决策有用的量"上——与 VZ 的 R-PE "PE 是 readout 而不是模型重建" 理念高度对应。
- **2405.12399 DIAMOND**：把世界模型从离散 latent 换成 **diffusion model**（EDM-based，1 步去噪也稳定），证明视觉细节对 RL 重要；Atari 100k mean human normalized 1.46，且能作交互 neural game engine（Counter-Strike: GO）。命中 **R3（diffusion latent 也是 z_t 一种参数化）/ R-PE（去噪即 PE 的连续松弛）**——给 VZ ReflectionEngine 的 imagination rollout 提供生成式备选。
- **2403.00564 EfficientZero V2**：MuZero 系的 sample-efficient 工程化，扩展到连续控制 + 视觉低维统一框架；**66 任务 50 任务超越 DreamerV3**（spotlight ICML 2024）。命中 **R3（model-based planning）**；其 limited-data 优势对 VZ 这种"从对话日志学习"的样本受限场景直接相关——尽管 VZ 不打算端到端训 substrate，但其搜索-学习配比经验可借鉴。
- **2503.18938 AdaWorld**：从 video 自监督提取 **latent actions**（捕捉 frame 间关键转移），autoregressive WM 以 latent action 为条件；新环境少量交互即可适配，跨异构动作空间。命中 **R3（latent action = action-channel z_t）/ R4（动作不在 token 而在学到的潜空间）**——这是"action channel 不直接来自硬编码"原则在视频域的最新实证；与 VZ 的"涌现优于硬编码"原则一致。
- **2410.24164 π₀ (Pi-Zero)**：在冻结 PaliGemma VLM 之上加 **flow matching action expert**，融合 7 robot platform / 68 task 的 dataset；可零样本 / 语言指令 / 高层 VLM policy / fine-tune 多模式工作；real-time 折叠衣物/装箱/打包等复杂操作。命中 **R2（VLM backbone 冻结）/ R3（action 用 flow 在 latent 生成）/ R4（control 不走 token）**——冻结基底 + 控制器侧 action head 的机器人版示范。
- **2301.08243 I-JEPA**：图像 SSL 不重建像素而**预测 representation 空间中的 target block**；ViT-Huge/14 + 16 A100 < 72 小时；不依赖 hand-crafted augmentation。**JEPA 范式之父**：所有 V-JEPA / V-JEPA 2 / Cwm 都建立在"predict in latent"上——LeCun 的 *A Path Towards Autonomous Machine Intelligence* 工程落地版；与 VZ R-PE "PE 在 latent 而非 surface" 形成最直接的工程映射。

### 备选（11-20）

- **2301.04104 DreamerV3**：固定 hyperparam 跨 150+ 任务超越调优 baseline；symlog + KL balancing + free bits；首个 from-scratch 收集 Minecraft 钻石的算法；NeurIPS 2025 Outstanding。
- **2206.14176 DayDreamer**：Dreamer 上真机：A1 四足 1 小时学走、UR5/XArm 抓取、Sphero 视觉导航——证明 model-based RL 不需要 sim2real，直接物理世界 sample-efficient。
- **2404.08471 V-JEPA**：视频 mask denoising in latent；VideoMix2M 200 万视频；Kinetics-400 81.9 / SSv2 72.2；frozen backbone 通用。
- **2510.02387 CWM**：32B 开源 LLM；mid-train 在 Python interpreter + Docker agent 轨迹上学 observation-action 关系；SWE-bench Verified 65.8%——**世界模型概念扩展到代码执行域**。
- **2501.03575 Cosmos**：NVIDIA Physical AI 平台级 WFM（Predict / Transfer / Reason 三件套 + 视频 tokenizer + curator + guardrails）；diffusion + autoregressive 双轨，开放权重。
- **2310.06114 UniSim**：把多源（图像 / 机器人 / 导航 / 人类视频）数据 orchestrate 成统一交互模拟器，high-level "open the drawer" 与 low-level "Δx, Δy" 双层动作；ICLR 2024 outstanding。
- **2405.15223 iVideoGPT**：autoregressive transformer 把 video / action / reward 都 tokenize；compressive tokenization；百万机械操作轨迹预训。
- **2406.09246 OpenVLA**：7B 开源 VLA = SigLIP+DINOv2 + LLaMA-2 7B；970k OpenX-Embodiment；29 task 超 RT-2-X (55B) 16.5pp；fine-tune via LoRA、8-bit serving 都行。
- **2309.17080 GAIA-1**：Wayve 9B 驾驶 WM，4700 小时英国驾驶数据；video / text / action 多模态 conditioning；可生成"红绿灯变色"、"车撞过来"等可控反事实场景。
- **2504.16054 π₀.₅**：π0 升级版，跨高层语义预测 + web data + 异构示例 co-training；陌生家庭长时段清洁/整理 demo。

## 候选→入选筛选规则的几条具体应用

1. **VZ=5 优先**：#1-#4 四篇全部 VZ=5。其中 #2 V-JEPA 2 是"冻结视觉基底 + 控制器侧 action-conditioned predictor"的最直接外部对位，#1 Dreamer 4 是 imagination training 的工程化范例，#3 Genie 是 latent action interface，#4 RLVR-World 是 RL-trained WM 路径——四篇覆盖了 VZ 在 A2 轴的四个核心借鉴方向。
- **R2 冲突的纯端到端方案降权**：Dreamer V3 / Genie / SIMA 2 / Cosmos / GAIA-1 均训练巨型 latent backbone 与 R2 冻结基底有路径分歧；它们仍入选是因为方法学（latent action interface、imagination training、verifiable RL training）对 VZ 控制器层有迁移价值——参考 [`research/deepmind-author-paper-assessment-2026-05.md`](../../deepmind-author-paper-assessment-2026-05.md) §4.3 对此差异的明确说明。
3. **classic 上限**：2 篇经典（#5 MuZero / #10 I-JEPA）远低于 4 篇上限。两篇都是"今天还要看"——MuZero 是所有现代 model-based 学习算法的语法本，I-JEPA 是 JEPA 范式之祖。
4. **DeepMind ≤ 4**：Top 10 中 DM = 3（#1 Dreamer 4 / #3 Genie / #5 MuZero）；11-20 中只补 1 篇（虽然 Dreamer V3 也有 DM 共著但主作者从 Toronto / Berkeley 视角看），仍守住单机构上限。Meta FAIR Top 10 = 2（#2 / #10），11-20 += 2（#13 / #14）= 总 4，正好至 Meta 上限。Tsinghua Top 10 = 2（#4 / #7），11-20 += 1（#17）= 3。
5. **中国厂内代表**：Tsinghua 系（RLVR-World #4 + EfficientZero V2 #7 + iVideoGPT #17 = 3 篇）+ 港科大/西湖（AdaWorld #8 = 1 篇）= **4 篇中国学界 / 厂前研究**贡献，符合 [`research/probe/01_method_and_scoring.md`](../01_method_and_scoring.md) §3 "中国厂内整体不少于 8 篇"在本轴贡献份额。
6. **Cross-axis 对照说明**：本轴中 EvoAgent (Y) 落在 A3 continual learning，DMWM / WorldLLM (Y) 落在 R-PE，SIMA 2 (Y) 落在 B3 open-ended，Embedded UPI (Y) 落在 R7 dual-track；它们与 A2 的耦合在精读卡阶段（[`research/probe/notes/A2/`](../notes/A2/)，待生成）将被作为跨轴交叉链接，**不重复计入** A2 Top 10 / 11-20。
