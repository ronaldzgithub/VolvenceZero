## 各论文一句话定位（10 篇 + 备选若干）

### Top 10 入选

- **2502.05171 Recurrent Depth**：在隐空间循环递归同一 transformer block + 自适应步数（adaptive ponder），无需输出新 token 即可加深推理算力，3.5B 参数 + recurrent depth 在 ARC/GSM 上接近 50B 模型 — 命中 **R3（z_t 控制器空间字面实现）/ R4（内部计算与 token 表达解耦）**。
- **2412.06769 Coconut**：把上一步 final hidden state（"continuous thought"）回写为下一步 input embedding，让 reasoning chain 在连续潜空间内推进，绕开离散 token bottleneck；GSM8k / ProsQA 上以更少 token 超 CoT — 命中 **R3 / R4**，"在控制器代码 z_t 空间做推理"的最直接 LLM 落地。
- **2403.09629 Quiet-STaR**：每个 token 位前生成隐 rationale，用 down-stream 下一 token 预测改善作为 REINFORCE 信号自学"先想再说"，泛化到任意 web text；在 GSM8k zero-shot 提升 5.9 → 10.9 — 命中 **R3 / R4 / R-PE（隐推理价值由后续 PE 反向背书）**。
- **2305.20050 Let's Verify Step by Step**：用 80 万人工 step-level 标签训练 PRM（PRM800K），best-of-N 在 MATH 上击穿 ORM 上限（78.2% vs 72.4%）；首次给出 step-level reward 工程根基 — 命中 **R-PE**（step-level PE = 过程奖励信号）。**今天还要看**：所有后续 process reward / R1 / Math-Shepherd / SCoRe 的根；R-PE 不是孤立洞见而是已有 frontier-lab 工程基线。
- **2409.12917 SCoRe**：双阶段多轮 online RL + reward shaping（regularize 第一回合不偏离 base + 增加第二回合改对 bonus），让模型从自生成数据学会真实 self-correct 而非行为坍缩；Gemini 1.5 Flash 在 MATH 上自纠 +9.1pp — 命中 **R-PE（用自身 PE 修正） / R3（multi-turn correction 是 z_t 切换的初等形式）**。
- **2501.12948 DeepSeek-R1**：直接对 base model 用 GRPO（无 SFT 冷启动）跑出长 CoT、self-verification、aha 时刻等推理行为；R1-Zero 在 AIME pass@1 从 15.6 → 71.0；多阶段冷启动 + 拒绝采样得到 R1，蒸馏到 1.5B-70B 链 — 命中 **R3（推理深度涌现）/ R-PE（verifier 奖励驱动），但 token 空间 RL 反例需注意 R4 边界**。
- **2312.08935 Math-Shepherd**：用 MC rollout 终态正确率自动给中间步打软标签（hard estimation + soft estimation 两版），免人工 step 标注训练 PRM；step-level value 也用作 step-PPO 的 reward，比 ORM 收敛快 — 命中 **R-PE（PE 自动归因，把"答案 PE"反向蒸馏成"步骤 PE"）**。
- **2408.03314 Snell test-time scaling**：系统比较 process-reward search vs adaptive sample-revision 两类 test-time 方法，给出"FLOPs-equivalent 下何时 14× 小模型 + 多 inference > 大模型 + 少 inference"的计算最优规律 — 命中 **R3（计算分配 = z_t 步数预算）**；为 metacontroller 的"该思考多久"决策提供量化曲线。
- **2501.19393 s1**：用 1K 高质量 reasoning trace SFT + 推理时强制追加"Wait"延长思考的 budget forcing，仅 1K 数据 + 32B 模型达到接近 o1-preview frontier；同时给出 monotonic scaling curve — 命中 **R3（test-time budget 即 z_t 步数门控的极简版）**。
- **2203.14465 STaR**：用 few-shot prompt 生成 CoT，正确者 fine-tune 入 weight；错的题给答案后回写 rationale；迭代自举推理；GSM8k 6 → 28 — 命中 **R3 / R-PE 的最早闭环**。**今天还要看**：所有后续 RL+verifier（DeepSeek-R1、ReST、ReSTEM、Quiet-STaR）的祖宗；理解 reasoning 自举的不变量必读。

### 备选（11-20）

- **2305.10601 Tree of Thoughts**：把推理建模成树搜索 + state evaluator + BFS/DFS；Game24 4 → 74。Token 空间搜索代表，违反 R4。
- **2501.12599 Kimi k1.5**：Moonshot 长上下文 + 多模态 RL reasoning，与 R1 同期，工程报告但实现细节有限。
- **2308.09687 Graph of Thoughts**：ToT 的有向图泛化，允许节点合并/聚合，工程贡献为主。
- **2303.11366 Reflexion**：LLM agent 用 verbal feedback + episodic memory 重试，HumanEval 80 → 91；token 空间 critique。
- **2203.11171 Self-Consistency**：CoT 多次采样取多数答案；GSM8k +18pp。简单稳健的 baseline，被 R1 + PRM 整合。
- **2303.17651 Self-Refine**：自己生成 critique 再迭代 refine，无需训练；7 任务平均 +20%。Token 层 critique 经典。
- **2312.06585 ReSTEM**：EM 自训练（采样 → 过滤 → SFT），被证 self-data > human data；DeepMind PaLM-2 实证。
- **2308.08998 ReST**：offline RL grow + improve 步法；机器翻译反馈最大化。R1 后被 GRPO 上位。
- **2407.21787 Large Language Monkeys**：Inference compute = repeated sampling，给出 coverage = c·log(N) scaling law；OSS small model + 200 sample 可超 GPT-4o。
- **2505.03335 Absolute Zero Reasoner**：self-play proposer + solver 自生成可验证 task；零外部 reasoning data 在 code/math 上达 SOTA。R-PE 自驱探索范式。

  - (d) **Self-consistency / refinement / self-critique** = 1 篇 Top 10（#5 SCoRe）+ 备选 #14/15/16 三篇经典
  - (e) **RL + verifier** = 2 篇（#6 DeepSeek-R1, #10 STaR）+ 备选 #17/18/20

## 候选→入选筛选规则的几条具体应用

1. **VZ=5 优先**：#1-#4 四篇全部 VZ=5（直接落点 R3/R4/R-PE 接口），破格放宽其他维度的考虑（如 #1 Recurrent Depth 影响力暂为 4 仍排第 1）。
2. **token 空间搜索一律降权**：ToT/GoT 影响力高但 VZ ≤ 3，留备选不入 Top 10。
3. **classic 上限**：3 篇经典（#4 / #7 / #10）紧贴 4 篇上限，全部为后续 frontier 工作的根，"今天还要看"已在一句话定位中说明。
4. **DeepSeek 不超过 4**：DeepSeek-R1 + Math-Shepherd（合著） = 2，远低于上限，仍可在备选 Kimi 上扩。
5. **OpenAI/DeepMind 不偏置**：OpenAI 1 篇（#4 Let's Verify）/ DeepMind 1-2 篇（#5 SCoRe + #8 Snell 合著），与 Stanford 3 / DeepSeek 2 / Meta 1 / Tübingen 1 / Berkeley 1 形成均衡谱。
