- **PDF**：⏳ 待下载到 `research/probe/papers/A4/opal-offline-primitives-2010.13611.pdf`
- **作者/venue**：Anurag Ajay / Aviral Kumar / Pulkit Agrawal / **Sergey Levine** / Ofir Nachum（MIT+Berkeley+Google）— ICLR 2021
- **子领域**：(e) Compositional skill / behavior basis — offline primitive discovery
- **四维评分**：理论新颖度 **4** / 工程深度 **4** / 影响力 **4** / VZ 关联度 **4**
- **R-ID 命中**：R5 **补全**（从 offline 经验离线发现 primitive，与 episodic memory 复用同源）；R3 **加强**（离线发现的 primitive 作为下游 z_t 的 prior）
- **核心贡献**：用 conditional VAE 从大规模 unlabelled offline 数据中提取 short-horizon **continuous primitive**（latent z），下游 RL 直接在 primitive 空间执行；offline RL 与 imitation learning 的样本效率显著提升。
- **行动建议**：**可立刻反哺**——给 [`docs/specs/continuum-memory.md`](../../../docs/specs/continuum-memory.md) 与 [`docs/specs/temporal-abstraction.md`](../../../docs/specs/temporal-abstraction.md) 提供"如何离线从历史日志中萃取 primitive，下游 z_t 在 primitive 空间运作"的具体算法；与 R-PE 离线反思（background-slow）天然契合。
- **既有 PDF**：⏳ 需新增下载。

## 3. 跳过 / 不入选（8 篇 — 含原因）

> 这些都是 A4 轴下被检索/讨论过的候选，但因强相关替代、与轴定义偏离、或哲学冲突而**不入候选池**。每条注明跳过原因，避免后续重复审议。

### 3.1 VIC — Variational Intrinsic Control (`1611.07507`)

- **作者/venue**：Karol Gregor / Danilo Rezende / Daan Wierstra（DeepMind）— 2016 arXiv
- **跳过原因**：DIAYN（§2.1）是 VIC 的直接改进版，引入 fixed prior 并解决 VIC 的 mode collapse；今天的 skill discovery 综述统一引用 DIAYN，VIC 仅作为历史背景。**用 DIAYN 代表 (b) 子领域足矣**。

### 3.2 EDL — Explore, Discover and Learn (`2002.03647`)

- **作者/venue**：Víctor Campos et al. — ICML 2020
- **跳过原因**：在 DIAYN 基础上增加 state-coverage 约束（避免 skill 集中在 state 空间小区域），增量贡献；已被 APT/APS 谱系吸收。**优先选 DIAYN 经典**。

### 3.3 HIRO — Data-Efficient Hierarchical RL (`1805.08296`)

- **作者/venue**：Ofir Nachum / Shixiang Shane Gu / Honglak Lee / **Sergey Levine**（Google Brain）— NeurIPS 2018
- **跳过原因**：feudal goal-conditioned 双层 HRL，思路与 FuN（§2.4）同源；HIRO 在连续控制（机器人）上更强，FuN 在 Atari/DM Lab 更强。**FuN 选作 (c) 代表（DeepMind 谱系完整 + 引用更高）**；若未来 VZ 关注连续控制，可补 HIRO。

### 3.4 h-DQN — Hierarchical-DQN (`1604.06057`)

- **作者/venue**：Tejas D. Kulkarni / Karthik R. Narasimhan / Ardavan Saeedi / Joshua Tenenbaum（MIT/Harvard/Google）— NIPS 2016
- **跳过原因**：HRL 早期工作，已被 FuN/HIRO 在多数 benchmark 上 superseded；其"显式 subgoal 集合"硬编码与 R3 涌现优先原则冲突。**仅作历史背景**。

### 3.5 HAC — Hierarchical Actor-Critic (`1712.00948`)

- **作者/venue**：Andrew Levy et al. — ICLR 2019
- **跳过原因**：三层 HRL with hindsight transitions；niche，多数后续工作回到双层（FuN/HIRO/MANGO）。**与轴 A4 主线相关性弱于其他候选**。

### 3.6 APS — Active Pretraining with Successor Features (`2108.13956`)

- **作者/venue**：Hao Liu / **Pieter Abbeel**（Berkeley）— ICML 2021
- **跳过原因**：把 APT（state entropy maximization）与 SF 结合做 pretraining，增量贡献；**DIAYN（MI-based skill discovery）+ SF for Transfer（SF/GPI 经典）分别覆盖更清晰**，APS 作为组合算法不必单独入选。

### 3.7 π0 — A Vision-Language-Action Flow Model (`2410.24164`)

- **作者/venue**：Kevin Black et al. (Physical Intelligence)
- **跳过原因**：robotics-specific VLA flow model，latent action 仅是其架构副产品；主轴是 A2（World Models）/ B3（embodied scaling），与 A4 "时间抽象 / option / skill" 主轴关联弱。**已在 A2 候选池讨论**。

### 3.8 ASAP — Aligning Simulation and Real Physics for Humanoid (`2502.01143`)

- **作者/venue**：Tairan He et al.（CMU+Berkeley+NVIDIA）
- **跳过原因**：humanoid 真实物理对齐，sim-to-real 工程主线；与 A4 时间抽象 / skill 概念无直接关系，是另一类问题。**轴外，跳过**。
