## 各论文一句话定位（Top 10 + 备选若干）

### Top 10 入选

- **2501.00663 Titans**：把 attention 视为短期上下文记忆，把可在 test-time 用梯度更新的 neural memory 模块视为长期记忆，三种 wiring（MAC / MAG / MAL）将两者组合，BABILong 2M 上下文 + needle-in-haystack 显著优于 Mamba/Transformer baseline —— 命中 **R5（记忆连续谱）/ R6（test-time 写入即在线巩固）**；可作为 CMS persistent stratum 的 update rule 直接参考。
- **2504.13173 Miras (It's All Connected)**：将 Transformer / Titans / 现代线性 RNN 全部视为同一类 **associative memory module**（attentional bias objective + retention regularization），forget gate ≡ retention 正则项；提出 attentional bias 的 4 种损失族 —— 命中 **R5 / R6**，是把"优化器=记忆=架构"统一在一个语言下的关键桥梁论文，最适合写进 CMS 术语表。
- **2601.09913 CMA (Continuum Memory Architectures)**：把 CMA 定义为**行为级类**（不绑定实现），提出 6 条必要充分条件（Persistence / Selective retention / Retrieval-driven mutation / Associative routing / Temporal continuity / Consolidation+abstraction），每条对应一个认知科学源头 + 操作化判据；4 个行为探针 + permutation test 证明 CMA 决定性胜 RAG 82-vs-10 —— 命中 **R5 / R6 完整同构**。VZ 与该论文已经过 [`D1_continuum_memory_deep_analysis.md`](../../openai-frontier-2026/notes/D1_continuum_memory_deep_analysis.md) 深度对照，6 条要求中 VZ 满足 4 条完整 + 2 条部分；可借鉴的两个具体机制（spreading activation engine + 4 行为探针套件）已落到 A11/A12 行动项。
- **2312.00752 Mamba**：选择性状态空间模型，让 SSM 参数依赖输入即可基于内容选择性遗忘/保留；硬件感知并行算法在 recurrent 模式实现 5× 吞吐 + 线性长度扩展；在语言、音频、基因组上均超越同尺寸 Transformer —— 命中 **R5（fixed-state 是工作记忆的极简实现）/ R2（substrate 替代候选）**。是 vz-substrate 后续 open-weight backbone 调研无法绕开的根。
- **2502.12110 A-Mem**：Zettelkasten 式动态卡片系统：每条新记忆自动生成 keywords / tags / contextual description，新卡链入时 LLM curator 给老卡更新属性形成有机演化网络；6 个长程对话基准 MRR/F1 平均 +20% —— 命中 **R5（episodic→semantic 演化路径）/ R6（writeback 触发结构演化）**。**注**：与 NL/ETA "记忆即架构、更新规则被学到" 直接冲突的 LLM-as-curator 风险已在 [`core-author-paper-assessment-2026-05.md`](../../core-author-paper-assessment-2026-05.md) 第七节标注，不可整体移植，仅借鉴 attribute schema。
- **2502.14802 HippoRAG 2**：以海马体 indexing theory 为灵感，PageRank + 个性化稠密召回 + deep passage integration，多跳 QA 上以 10-20× 更便宜 / 6-13× 更快接近 IRCoT；NeurIPS 2024 —— 命中 **R5 / R6**。NL/ETA 裁判线下仅其多跳算法片段可作为 DerivedRetrievalIndex 的 owner-internal 召回算法可选项，不引入第二 memory owner。
- **1612.00796 EWC**：用 Fisher 信息估计每个权重对旧任务的重要性，新任务训练时给重要权重加二次惩罚；在 MNIST permutations + 多 Atari 顺序学习上证明可避免灾难性遗忘；DeepMind / PNAS 2017，2k+ 引用 —— 命中 **R6（弹性巩固）/ R2（基底受约束更新）**。**今天还要看**：所有后续 weight-importance 系（SI / MAS / online EWC / EWC++）和 ModificationGate VC-bounded 路线（Two-Gate Guardrail）的根；为 vz-substrate adapter-delta 容量约束提供可参考的连续宽放回放替代。
- **2401.08623 Wake-Sleep Consolidated Learning**：仿生 wake-sleep 两阶段 + replay 防灾难性遗忘 —— 命中 **R6（background-slow 反思阶段算法雏形）**；与 VZ 的 session-post slow loop 写两类产物（记忆沉淀 + 策略沉淀）结构同构。
- **1410.3916 Memory Networks**：Weston/Chopra/Bordes ICLR 2015，定义外部记忆 + I/G/O/R 四组件框架，question answering 通过链式多 supporting fact 推理 —— 命中 **R5（外部记忆作为 dynamic KB 的最早形式化）**。**今天还要看**：所有 RAG / DNC / MemGPT / NTM / End-to-End Memory Networks / Sukhbaatar 2015 attention-on-memory 系的祖宗；理解"显式可读写记忆 ≠ 内部状态"的最早分界。
- **2509.16189 Latent Learning**：形式化论证参数学习不能完全代替 episodic memory（即使无参更新窗口长，episodic 灵活性仍是必需）；DeepMind —— 命中 **R5（连续谱中 episodic 层不可省）**，是 vz-memory 4-stratum 设计的合法性论文级背书。

### 备选 11-20

- **2505.23735 ATLAS**：批判固定大小 recurrent memory 的容量受限 / 仅最近 token 在线更新 / 管理表达力不足三类问题，给出 BABILong + 长上下文显著提升。是 Titans 之后的 capacity / management 升级版，可作 CMS 内部容量、压缩、retention policy 的细化参考。
- **2512.23852 Trellis**：固定大小 KV memory + test-time recurrent compression + forget gate，COLM 2025；为 vz-substrate residual / KV cache steering 与 bounded adapter-delta 入口提供"控制必须有容量边界"的成熟参考。
- **2511.07343 TNT**：粗粒度全局 + 多个局部高分辨率记忆的两阶段训练，解决 test-time memorization 的训练效率问题；适合 rare-heavy / offline refresh pipeline，不进入在线 turn path。
- **2601.02744 Synapse**：动态图 + 时间衰减 + 横向抑制 + 三种检索混合（spreading activation / structural / temporal），与 VZ derived 层多跳召回直接同构；spreading activation engine 价值已在 D1 落到 A11。
- **2512.22716 Memento 2**：把 episodic memory 当有读写操作的控制对象，不更新主模型参数即可持续适应；对应"冻结基底 + 记忆侧适应"的明示路线；备选位是因为 reflective writeback 与 VZ RFL1 现有实现重叠。
- **2405.21060 Mamba-2 (SSD)**：State Space Duality 框架揭示 Transformer ≈ SSM 的形式同构，2-8× 加速；增量于 #4 Mamba 但同子领域不重复占 Top 10 名额。
- **CLS Theory Updated（Kumaran/Hassabis/McClelland, Trends Cog Sci 2016）**：CLS 理论的 2016 更新版（hippocampus 快学具体 / neocortex 慢学结构 + replay goal-dependent reweighting + fast neocortical integration）；无 arXiv 但作为 R6 / vz-memory 4-stratum 的神经科学根据必引；已通过 #6 HippoRAG 2、#8 Wake-Sleep、#10 Latent Learning 间接吸收。
- **1410.5401 NTM**：Graves 2014，attention-based 外部 memory + read/write head + Turing-complete 可微计算 —— 与 #9 Memory Networks 同期奠基；机制被现代 attention 主流上位但术语（read/write head, content-based addressing）仍在用。
- **1703.04200 Synaptic Intelligence (SI)**：online 计算每个 synapse 对 loss 变化的贡献作为重要性，与 EWC 互补；ICML 2017。
- **2405.14831 HippoRAG (v1)**：海马体 indexing theory + KG + Personalized PageRank 多跳，OSU+Stanford NeurIPS 2024；v2 已对其在 passage integration / multi-hop 上完整 subsume，留作 v1 设计原意参考。

  - **MeMo / Mem0 / SimpleMem / Hindsight (4-network 架构)**：与 #5 A-Mem / #15 Memento 2 / #30 MemGPT 同 (b) agentic-memory 路线但 NL/ETA 裁判线（不创造第二 memory owner / 更新规则不被外置 LLM curator 替代 / 检索不引入并行嵌入空间）下与 VZ 现有 CMS 冲突更深，留作竞品对照不入主名单。
  - **MemoRAG (Qian et al. 2025)**：与 #6 HippoRAG 2 同 (b)(g) RAG-augmented memory 路线，但实证规模与方法新颖度均弱于 HippoRAG 2，CMA Related Work 已并列引用。
  - **GraphRAG (Microsoft 2024)、KV-cache RAG、RAGFlow 等工程报告**：偏 application/serving 工程，违反 [`01_method_and_scoring.md`](../01_method_and_scoring.md) 排除原则第 2 条（纯系统工程）。
  - **Pretrained transformers as universal computation engines、LongRoPE、YaRN 等长上下文外推**：偏 substrate 工程而非记忆机制；归 (d) 但 cognitive 现象不足，留作 vz-substrate 调研。
  - **Toolformer / AssistGPT / Reflexion 等 agent 系**：行为更接近 plan/credit 而非"记忆作为 owner"，归 A4/A5 而非 A3。
