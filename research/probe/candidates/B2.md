## 各论文一句话定位（10 篇 + 备选若干）

### Top 10 入选

- **2512.18202 Sophia (System-3 Persistent Agent)**：在冻结 LLM 之上加 meta-cognitive 层，4 个机制（thought curation / narrative memory / dynamic user-self models / hybrid reward）；recurring 任务推理步数 -80%、高复杂度成功率 +40% — 命中 **R7（双轨 World/Self 字面落地）/ R11（user_model + self_model 同时持久化）/ R14（narrative identity = 持久 regime）**。VZ 关联 = 5：直接是 dual_track + regime + semantic_state 的另一种工程实例化。
- **2506.03543 CogniPair (GNWT)**：Global Workspace Theory 工程化为 551 个 GNWT-Agent + 5 类 sub-agent（emotion / memory / social norms / planning / goal）+ 全局广播；Speed-Dating 数据集 72% attraction correlation、77.8% match prediction — 命中 **R8（多 owner + 公共快照总线）/ R11（每个 sub-agent 是一个 semantic owner）/ R16（多方身份）**。VZ 关联 = 5：与"快照隔离 + 多 owner"惊人地像（虽然他们更激进，全局广播 vs VZ 的有界传播）。
- **2502.14171 ToM-aligned BDI**：在 LLaMA-3 (3B/8B) 上把 BDI 三元组（Belief / Desire / Intention）显式抽出做对齐微调，win rate 67% / 63% — 命中 **R11（直接对应 plan_intent / commitment / open_loop / user_model 四类 owner 的拆分思路）/ R7（user 的 BDI 映射 World 轨道，self 的 BDI 映射 Self 轨道）**。VZ 关联 = 5：是 9 类 semantic owner 命名的最直接语言学先例。
- **2502.11881 ThoughtTracing**：类 SMC 粒子滤波，每步生成多个 belief 假设并用 likelihood 重采样，无需 ground-truth；在多个 ToM benchmark 上显著超 baseline，并揭示 o3 / R1 在隐式 ToM 上的盲区 — 命中 **R11（user_model 在线维护无监督）/ R-PE（行动 likelihood 即 social PE 信号）/ R16（多方信念分别 tracking）**。VZ 关联 = 5：user_model owner 的"在线无监督维护"算法蓝本。
- **2505.14685 Lookback Mechanism**：在 Llama-3-70B / 405B 残差流中实证发现 character-object-state 通过 Ordering ID + 低秩子空间绑定，分 binding / answer / visibility 三种 lookback；构造 CausalToM 因果数据集 — 命中 **R11（"内部状态可几何定位"硬证据）/ R4（控制器空间 ≠ token 空间，OI 是 z_t 的天然候选）/ R8（user_model 快照可对应到具体子空间）**。VZ 关联 = 5：是 R11 的最强机理可解释证据，可立刻反哺 user_model / belief_assumption owner 的内部表示设计。
- **2512.07092 Geometry of Persona (Soul Engine)**：基于 Linear Representation Hypothesis，把人格视为 frozen Qwen-2.5 隐空间中的**正交线性子空间**；dual-head + SoulBench；MSE 0.011，无 alignment tax，零样本 personality injection — 命中 **R14（regime 是几何对象不是 prompt 标签）/ R2（冻结基底 + 可加性人格 delta）/ R11（self_model 的几何表征）**。VZ 关联 = 5：把"regime 不是 prompt 标签"从设计原则升级为已被实证的几何事实。
- **2602.16301 Co-player Inference**：标准 decentralized RL + co-player 多样性即可让 sequence model 内涌现 best-response 与合作；不需要硬编码学习规则或时间尺度分离 — 命中 **R7（合作行为是涌现的，不是 prompt-driven）/ R16（多方身份/对手建模）/ R3（in-context 即 z_t 短时控制）**。VZ 关联 = 4：给 Self 轨道在多人场景中"涌现而非编码"的范式背书。
- **2508.12935 RLFF-ESC (Future-Oriented Rewards)**：用多 agent 模拟未来对话轨迹收 future-oriented reward → 训 reward model → 训 ESC policy；Qwen2.5-7B / Llama3.1-8B 在两个 ESC 数据集上稳定超基线 — 命中 **R-PE（长程社交 PE 不等于即时 PE）/ R7（relationship 轨道的延迟回报建模）/ R12（关系连续性评估而非单轮帮助）**。VZ 关联 = 4：与"长程关系不是当下 reward"完全合拍，可作为 self_temporal owner 的 reward shaping 模板。
- **2210.05492 No-Press Diplomacy (DiL-piKL / RL-DiL-piKL)**：把人类模仿 policy 当 KL 锚正则化自博弈 RL，证明在合作博弈中比纯 self-play 更稳；Diplodocus 在 200 局人机比赛中 Elo 排名第 1 / 第 3 — 命中 **R10（"人类正则化"即一种 ModificationGate 形态：自博弈不能脱离人类先验）/ R7（合作不是纯对抗 utility）/ R16（多方策略心智推理）**。VZ 关联 = 3 但影响力 = 5：CICERO 系列的算法心脏，给 Self 轨道在多方策略场景下提供"人类先验作为有界先验"的形式化路径，今天还要看是因为它把"strategic ToM + cooperation"统一成一个可量化的 RL 问题。
- **2306.15448 BigToM**：Stanford CICL 用因果模板程序化生成 5000 个 ToM 评估题（25 controls），人评质量 ≥ 众包 ≈ 专家；首次系统区分 GPT-4 vs 其他 LLM 的 ToM 能力差异 — 命中 **R12（评估覆盖 social cognition 而非任务）/ R11（评估正是用来读 user_model 是否 well-formed）**。VZ 关联 = 3 但影响力 = 4：今天还要看是因为 BigToM 是后续所有 ToM benchmark（FANToM / ExploreToM / SimToM）的程序化生成模板源头，VZ 评估族 6（interaction quality）需要它做 ToM 维度基线。

### 备选（11-20）

- **2310.15421 FANToM**：信息非对称多方对话上的 ToM 压力测试（10K 题 / 256 对话）；揭示 LLM 在简单 belief 题上做对、复杂 answerability 题上失败 → 揭示 illusory ToM。EMNLP 2023。
- **2412.12175 ExploreToM**：Meta + UW + CMU 用 A* + DSL 对抗性生成 ToM 故事；Llama-3.1-70B / GPT-4o 准确率仅 0-9%；fine-tune 后 ToMi +27pp。ICLR 2025。
- **2510.07925 Personalized Long-term LLM Interactions**：持久记忆 + 动态协调 + 演化用户档案；与 user_model + relationship_state 同构。已有 Y。
- **2512.06688 PersonaMem-v2**：隐式人格学习 + agentic memory + 128k 多轮 + 16× 上下文压缩。已有 Y。
- **2509.25299 ID-RAG**：把人格固定成结构化 KG（核心信念 / 特质 / 价值），给 regime 一个具体落地。已有 Y。
- **2512.12716 CoDA**：单 LLM 同时演 Planner / Executor + context 隔离 + PECO 联训。已有 Y。
- **2506.16756 SocialSim**：情感支持对话生成数据集 + 模拟器。已有 Y。
- **2508.09521 COMPEER**：把心理学帮助技能（Helping Skills）当 RL 过程奖励维度。已有 Y。
- **2410.18636 Learning-Aware Policy Gradients**：opponent learning aware（LOLA 后裔）的多 agent 合作。已有 Y。
- **1802.07740 ToMNet (Rabinowitz)**：DeepMind ICML 2018 用 meta-learning + 行为观察学他者心智模型；通过 Sally-Anne。今天作为"machine ToM 起点"参考。

## 候选→入选筛选规则的几条具体应用

1. **VZ=5 优先且爆量**：B2 是最贴近 VZ "关系 + 主体性 + 9 类 owner" 哲学的轴；#1-#6 全部 VZ=5。即 **6/10 都是直接落点**，远超 A1 的 4/10。这是因为 R7 / R11 / R14 / R16-R20 的语义 owner 谱系刚好与 ToM / persona / belief tracking / multi-party identity 文献一一对应。
2. **机理可解释优先**：#5 Lookback 给"R11 内部状态可几何定位"提供机理证据，#6 Geometry of Persona 给"R14 不是 prompt 标签"提供线性子空间证据 — 两篇组合起来，VZ 的几何身份观从设计假设升级为**已被验证的实证规律**，应作为 P0 立刻反哺。
3. **CN 机构刻意均衡**：#1 Sophia (Westlake/SJTU) + #6 Geometry of Persona (Capital Medical Beijing) + #8 RLFF-ESC (HKBU/Tencent) = 3 篇 CN 主导，加上备选 #14 / #17 / #18 的 CN 来源，本轴对"中国厂内 ≥ 8 篇跨轴聚合"贡献 ≥ 3。
4. **避免单一机构集中**：Stanford 在 #4 ThoughtTracing（合著）+ #10 BigToM（主导）= 2 处出现；MIT 在 #4 + #5 = 2 处出现；Meta 在 #9（主导）= 1；Google 在 #7（主导）= 1。所有单机构 ≤ 2，远低于 ≤ 4 上限。
